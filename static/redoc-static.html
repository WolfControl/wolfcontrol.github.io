Get sensor readings from a gateway

Select by time range and optionally node Ids

path Parameters
name
required
string

The Id of the gateway to retrieve data from.

query Parameters
nodeIds
Array of strings

An array of metric Ids to retrieve. Optional, if not provided, all metrics for the gateway are returned.

startTime
required
string <date-time>

The start of the time range for filtering sensor data (inclusive).

endTime
string <date-time>

The end of the time range for filtering sensor data (inclusive). If not provided, defaults to the current time.

Responses

https://app.wolfcontrol.io/api/v1/gateways/{name}/metrics
